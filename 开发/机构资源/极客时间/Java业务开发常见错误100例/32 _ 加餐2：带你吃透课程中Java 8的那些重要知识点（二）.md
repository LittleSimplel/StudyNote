<li>利用flatMapToDouble方法把列表中每一项展开替换为一个DoubleStream，也就是直接把每一个订单转换为每一个商品的总价，然后求和。</li>
</ul><pre><code>//直接展开订单商品进行价格统计

        .flatMapToDouble(order -&gt;
